使用帮助

  1. 首先需要利用API进行“增加一门课程”操作(在“dashboard”中有相关按钮)，或者使用系统里目前已经增加的课程进行操作
  2. 按照顺序，执行“录入知识点”，“数据抓取”，“依赖关系分析”，“人工校验”，“最终数据存储到图数据库”
  3. 最后，在“图谱可视化测试”中可以看到结果
  4. 如果想重新开始构建图谱，在“dashboard”中点击该课程的“讲此门课程的状态重置”
