使用帮助

  1. 首先在“dashboard”页面中“增加一门课程”（只需增加一次，该动作之后会在教师管理系统里作为一个按钮，然后进入图谱的后台管理系统）
  2. 按照顺序，执行“录入知识点”，“数据抓取”，“依赖关系分析”，“人工校验”，“最终数据存储到图数据库”
  3. 最后，在“图谱可视化测试”中可以看到结果
